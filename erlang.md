title: Erlang

* All statements must end with a period and whitespace
* Atoms are like Ruby symbols

## Shell

```erlang
help().
q().

f(Variable). % drop a variable
f(). % drop all variables
```

## Syntax cheatsheet

```erlang
% division
5 / 2.
5 div 2.
5 rem 2.

% base conversion
2#1010. % is base 2 for 10

% equality
1 =:= 1. % returns true
1 =/= 0. % returns true
```

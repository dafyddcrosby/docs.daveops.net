====== IPv4 - subnetting ======


===== Natural mask =====

| First bits of address | default mask | Decimal |
|-----------------------|--------------|---------|
| 0 x x x               | 8 bits long  | < 128   |
| 1 0 x x               | 16 bits long | 128-191 |
| 1 1 0 x               | 24 bits long | 192-223 |
| 1 1 1 0               | multicast    | 224-239 |

Greater than 239 the address is reserved

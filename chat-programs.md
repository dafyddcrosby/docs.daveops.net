# Hipchat


## Message commands

| command    | description         |
|---------- |------------------- |
| /part      | leave room          |
| /available | green bubble        |
| /away      | yellow bubble       |
| /dnd       | red bubble          |
| /topic     | set room topic      |
| /quote     | monospace font      |
| /code      | syntax highlighting |
| /me        | emote command       |


## Keyboard shortcuts

| shortcut    | action                         |
|----------- |------------------------------ |
| cmd+f       | search                         |
| cmd+i       | invite                         |
| cmd+t       | join room                      |
| cmd+shift+n | create room                    |
| cmd+w       | close room                     |
| cmd+n       | search for room, user          |
| option+up   | put last line in your text box |

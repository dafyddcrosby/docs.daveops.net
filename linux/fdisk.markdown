---
title: fdisk
---

```bash
# show partition table from CLI
fdisk -l <disk>
```

| description            | command |
|------------------------|---------|
| show menu              | m       |
| show partition table   | p       |
| create a new partition | n       |
| change filesystem type | t       |
| list filesystem types  | l       |
| write partition table  | w       |

Use `sfdisk` to script disk formatting
